Pasos para realizar la activivdad
se abre consola
se ejecuta comando 'pwd'
se ejecuta mkdir 'ejercicios' para crear carpeta nueva
se ejecuta comando cd 'ejercicios' para ingresar a esa nueva carpeta
se ejecuta comando 'code .' para abrir VScode
se sigue la ruta 'Archivo/crear carpeta'
se nombra 'ejercicio1'
se ejecuta 'touch README.md' para crear archivo en blanco
para configurar usuario y contrasena se ejecuta 'git config --global user.name
